
### Exemplo Rápido:

```python
from core.brain import KlipzaBrain

# Conecte o cérebro na sua API favorita
brain = KlipzaBrain(api_key="SUA_CHAVE", ai_type="openai")

# Deixe o cérebro pensar
resposta = brain.think("E aí, como tá o movimento hoje?")
print(resposta)
```
